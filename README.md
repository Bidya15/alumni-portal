# Alumni Portal

A comprehensive, state-of-the-art platform for managing institutional alumni networks, professional networking, and career engagement.

## Features
- **Profile Management**: Secure registration with OTP verification.
- **Career Hub**: Job postings, referral requests, and mentorship paths.
- **Events & Networking**: Dynamic event registration and alumni connections.
- **Administrative CMS**: Full control over site content, stats, and banners.

## Tech Stack
- **Backend**: Java 21, Spring Boot 4.0.3, PostgreSQL.
- **Frontend**: React 19, Vite, Framer Motion.
- **Security**: JWT-based stateless authentication.
